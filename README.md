## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npm run android // android app
   npm run ios // ios app
   npm run web
   ```

## Get a fresh project

When you're ready, run:

```bash
npm run reset-project
```
